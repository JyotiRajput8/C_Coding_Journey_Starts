# 💻 C Program: Hello Visual Studio

This is my first C program that prints welcome messages on the screen. It helps in understanding the basic structure of a C program and how output is displayed using the `printf()` function.

## Concepts Used

* Header Files
* `stdio.h`
* `main()` Function
* `printf()` Function
* Program Execution
* `return 0`

## Program Structure Explained

### Header File

```c
#include <stdio.h>
```

The `stdio.h` header file provides standard input and output functions. It allows us to use functions such as `printf()` and `scanf()`.

### Main Function

```c
int main()
```

The `main()` function is the entry point of every C program. Program execution starts from this function.

### Printing Output

```c
printf("Hello Visual Studio\n");
```

The `printf()` function is used to display text on the screen.

### Return Statement

```c
return 0;
```

The `return 0` statement indicates that the program executed successfully without any errors.

## Output

```text
Hello Visual Studio
I am here to learn C programming with you.
Please guide me.
```
